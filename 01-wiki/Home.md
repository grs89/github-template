# Home

Bienvenido a la documentación técnica del GitHub Template.

## Arquitectura

Este template incluye funcionalidades adaptadas de GitLab CI/CD para GitHub Actions.

### Orquestadores

- **docker-brk.yml**: BuildKit premium con push nativo
- **docker-matrix.yml**: Soporte para monorepos
- **docker-dnd.yml**: Docker-in-Docker estándar

### Seguridad

- SLSA Provenance
- SBOM CycloneDX
- Firma con Cosign
- Escaneos con Trivy
